dnyScript ist eine leichtgewichtige Skriptsprache, welche genutzt werden kann, um Aufgaben einer Anwendung
in Skripte auszulagern. Der Interpreter ist in C++ geschrieben und besteht aus zwei Quellcode-Dateien (Header und 
Source File), welche in einem eigenen C++ Projekt genutzt werden können.<br/><br/>
Die Skriptsprache unterstützt einfache Strukturen, wie z. B. Funktionen, Schleifen, Variablen und Kontrollstrukturen. <br/><br/>
Weiterhin gibt es auch eine Skript-Shell, welche man benutzen kann, um die Skriptsprache ein bisschen
auszuprobieren. Die Shell ist über Plugins erweiterbar, so bringt sie dadurch z. B. String-Manipulation oder
das Erstellen von Windows-Forms mit.<br/><br/>
Im folgenden mal ein Beispiel Code:<br/><br/>
<code>
	function foo int(bar int)<br/>
	{<br/>
	&nbsp;&nbsp;local res int;<br/>
	&nbsp;&nbsp;set res <= 0;<br/><br/>

	&nbsp;&nbsp;for (i, -ls, %bar) {<br/>
	&nbsp;&nbsp;&nbsp;&nbsp;+= res %i;<br/>
	&nbsp;&nbsp;};<br/><br/>

	&nbsp;&nbsp;result %res;<br/>
	};
</code>
Bitte konsultieren Sie die Readme im GitHub Repository, für Details, wie die Skriptsprache anzuwenden ist.<br/>
