Actifisys is a general activity finder platform system. You can use it to create your own activity
finder web service.<br/><br/>
Whether it is for finding BBQ activities, outdoor activities or board game activities - with Actifisys
you can set up a system for that particular purpose. Or just set up a general activity finder service
where users can search for any activity.<br/><br/>
Users on Actifisys create activities with a future date which are then listed in the activity feed. The
activities where the activity date is the smallest one will be shown first. But you can also filter
for a specific date range. Also users can specify a location which can be used for filtering, too.<br/><br/>
Users can comment on activites, participate or set themselves as potentially interested. Activities
can be restricted to a limit of participating people amount or can be restricted to a specific gender.
Participating people can also upload images, e.g. with activity related information or after the activity
some photos of the activity itself.<br/>
<br/>
Users can connect with each other by sending direct messages. Also users get e-mail and website notifications
on certain occasions<br/><br/>
