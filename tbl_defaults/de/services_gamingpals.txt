GamingPals.org ist eine internationale Plattform, auf der Gamer neue Spielerfreunde finden können.<br/><br/>
Dazu können sie eine Anzeige mit Details ihrer Anforderungen angeben. So lässt sich ein Titel, eine Beschreibung
sowie das Spiel und Land angeben. Ausserdem kann man auf Anzahl an Teilnehmer und Geschlecht beschränken.<br/><br/>
Interessierte Spieler können dann sich als Mitmacher oder Interessent setzen. Für einen Austausch lassen sich
innerhalb eines Gesuchs Kommentare verfassen, die als Diskussionsaustausch dienen. Man kann Tags spezifizieren,
die es ermöglichen, noch weitere Details anzugeben, über die andere Spieler das Gesuch finden können.<br/><br/>
Auf der Startseite hat man einen Feed, der alle aktiven Gesuche auflistet. Dieser lässt sich nach Region, Spiel,
Datum und Tag filtern.<br/><br/>
Es lassen sich auch private Nachrichten austauschen. Benachrichtigungen informieren die User über alle relevanten
Ereignisse.<br/><br/>
