dnyScript is a lightweight scripting language which you can use to outsource application tasks
in scripts. It is written in C++ and comes as two files, a header and a source file which you
can include in your C++ project.<br/><br/>
The scripting language supports basic operations such as variable assignment, if-elseif-else conditions,
loops and functions. <br/><br/>
There is also a scripting shell available which you can use to play around a bit with the scripting
language. The shell supports plugins which grants more functionality such as string
manipulation or creating Windows forms.<br/><br/>
An example code construct can look like as follows:<br/><br/>
<code>
	function foo int(bar int)<br/>
	{<br/>
	&nbsp;&nbsp;local res int;<br/>
	&nbsp;&nbsp;set res <= 0;<br/><br/>

	&nbsp;&nbsp;for (i, -ls, %bar) {<br/>
	&nbsp;&nbsp;&nbsp;&nbsp;+= res %i;<br/>
	&nbsp;&nbsp;};<br/><br/>

	&nbsp;&nbsp;result %res;<br/>
	};
</code>
Please refer to the repository readme for detailed explanations<br/>
