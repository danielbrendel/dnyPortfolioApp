Asatru PHP ist ein einfaches PHP Framework, welches ich entwickelt habe, um mein Verständnis von Frameworks zu verbessern.
Denn es ist eine Sache ein Framework zu benutzen - und eine andere Sache selbst eines zu erschaffen. Mir hat es wirklich
geholfen, meine Skills im Schreiben von PHP Code zu verbessern.<br/><br/>
Das Framework ist vor allem für einfache Dinge geeignet, wie Landingpages oder kleinere Dienste, da es nicht so
komplex wie ein etabliertes Framework wie Laravel oder Symfony ist. Es ist jedoch ein guter Start, da ich alle Features
abgedeckt habe, die ich bisher auch bei anderen Frameworks gebraucht habe.<br/><br/>
Das Framework lässt sich mittels Composer installieren:<br/>
<code>
	composer create-project danielbrendel/asatru-php
</code>
<br/>

Nachdem das Projekt erstellt wurde, geht man in den erstellten Projektordner und führt folgenden Befehl aus,
um die Anwendung zu starten:<br/>
<code>
	php asatru serve
</code>
<br/>
Dieser Befehl startet einen Webserver auf Port 8000. Jetzt können Sie zum <a href="http://localhost:8000">Localhost</a>
browsen.
Wenn alles geklappt hat, dann sollten Sie in Ihrem Browser folgenden Inhalt sehen:<br/><br/>
