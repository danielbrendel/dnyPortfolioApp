CorvusChat is a chat system where users can chat in channels or in privat conversations. Users can connect to chatservers
which can be dynamically configured by the administration. The chat server features a plugin system in order to extend the
server functionality. It also features an RCON system, so administrators can remote-manage their servers without connecting 
as a common chat client. Also there is a masterserver system which allows users to fetch a dynamic server list of available
servers.<br/><br/>

Chatrooms can be customized to fit the needs of the creator. For instance, you can set chatroom passwords and a topic which 
defines the theme of the chatroom.<br/><br/>

By default there is a C++ SDK available which you can use to code your own server plugins as DLLs which can extend the server
in various ways.<br/><br/>
