Astarlove is a dating plattform which you can use to create your own themed dating service. It is free to use and has almost no limitations.<br/><br/>
Users can send private messages without limit and costs. Only condition is that both chat partners have liked each other before.
This ensures that users don't get annoying or unwanted messages.<br/><br/>
Users have a search area where they can search for active members in their region. This search can be refined with filter options.
If you find someone interesting then you can send a like. If the user finds you interesting, too, then they can like you too and 
you can then start a conversation.<br/><br/>
The platform allows many profile settings to adjust, so you can create a detailed profile about yourself. Additional to you avatar you
can upload three more photos.<br/><br/>
Photos must be approved by the administration. This ensures that members do only see profiles with approbriate photos.<br/><br/>
