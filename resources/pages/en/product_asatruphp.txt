Asatru PHP is a lightweight PHP framework which I made to improve my understanding how a framework can actually work.
It is one thing to use a framework - and another thing to create one on ones own. For me it really helped me to
improve my skills in writing PHP code.<br/><br/>
Asatru PHP framework is especially suited for simple websites, e.g. landing pages or simple services. It is not as
complex as a full blown framework like Laravel or Symfony. Since my goal was to write a framework that covers everything
I need by other frameworks, it is at least a start.<br/><br/>
The framework is available via Composer:<br/>
<code>
	composer create-project danielbrendel/asatru-php
</code>
<br/>

After Composer has finished creating the project, cd into the project directory and issue the command:<br/>
<code>
	php asatru serve
</code>
<br/>
This will launch a webserver on port 8000. Now browse to <a href="http://localhost:8000">Localhost</a>.
If you see the following in your browser then the installation succeeded:<br/><br/>
