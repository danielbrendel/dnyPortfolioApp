This product is a tool which allows you to use various Unicode fonts in other applications.<br/><br/>

The tool allows you to type in text in the selected font in other applications directly, so there is no need
to enter the text first and then copy and paste it in the target application window.<br/><br/>

Note that in order for this to work, the target application must support UTF-16 encoding. If so, you can basically
use every available Unicode fontset.<br/><br/>

The tool features the dynamic adding of different fonts using alphabet files.<br/><br/>
